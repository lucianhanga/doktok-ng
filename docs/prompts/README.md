# Prompts

This folder holds the canonical handoff brief for DokTok NG.

- [`doktok-ng-original-brief.md`](doktok-ng-original-brief.md) — the original full specification used
  to kick off the project. It is the source of truth for product intent, the milestone plan, the
  ingestion pipeline, the database model, the security model, and the MCP strategy. The architecture
  proposal and ADRs in this repository are derived from it.

Note: the brief originally used the working name "DokTok2"; the project is named **DokTok NG**.
